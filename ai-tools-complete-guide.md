### Lesson 2.1: Automation Basics

**Core Concepts:**

1. **Trigger** - The event that starts the automation
   - Example: "New form submission"
   - Example: "New email received"
   - Example: "Product sold"

2. **Action** - What happens when triggered
   - Example: "Send email"
   - Example: "Create database entry"
   - Example: "Post to social media"

3. **Condition** - Rules that determine if action happens
   - Example: "IF amount > $100"
   - Example: "IF status = 'approved'"
   - Example: "IF time between 9am-5pm"

### Lesson 4.1: Prompt Engineering Basics

**The Formula for Effective Prompts:**

```
[ROLE] + [CONTEXT] + [TASK] + [CONSTRAINTS] + [OUTPUT FORMAT]
```

**Example 1: Product Description**

```
ROLE: You are an expert e-commerce copywriter
CONTEXT: I sell premium fitness equipment
TASK: Write a compelling product description
CONSTRAINTS: 150 words, highlight benefits not features, include emotional appeal
OUTPUT FORMAT: Markdown with headers

Product: Adjustable Dumbbell Set (5-50 lbs)
Target audience: Home gym enthusiasts
Key benefit: Space-saving, versatile
```

**Example 2: Business Analysis**

```
ROLE: You are a business strategy consultant
CONTEXT: I run a SaaS company with $100k MRR
TASK: Analyze my growth opportunities
CONSTRAINTS: Focus on 90-day wins, realistic, actionable
OUTPUT FORMAT: Numbered list with implementation steps

Current metrics: 100 customers, $1000 avg value, 10% churn
```
