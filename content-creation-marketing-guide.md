## MODULE 1: VIRAL CONTENT FORMULAS

### Lesson 1.1: The Hook-Story-Payoff Framework

**What You'll Learn:**
The exact formula used by top creators to make content go viral.

**The Tool: ChatGPT + Claude**

**The Framework:**
- **HOOK** (First 3 seconds) - Stop the scroll
- **STORY** (Middle) - Build curiosity
- **PAYOFF** (End) - Deliver value

**Prompt Template:**
```
Create viral content about [TOPIC] using the Hook-Story-Payoff framework.

Hook: Start with a surprising stat or question that makes people stop scrolling
Story: Tell a relatable story or example (2-3 sentences)
Payoff: Deliver the key insight or actionable tip

Format for: [PLATFORM - Twitter/LinkedIn/TikTok]
Tone: [TONE - Professional/Casual/Humorous]
```

**Real Example:**
```
Create viral LinkedIn content about productivity using Hook-Story-Payoff.

Hook: "Most people waste 3 hours per day on busy work"
Story: Tell about a CEO who discovered this and changed everything
Payoff: Share the 3 habits that saved her 15 hours per week

Format for: LinkedIn
Tone: Professional but conversational
```

**Expected Results:**
- 3-5x higher engagement
- More shares and comments
- Higher click-through rates

### Lesson 1.2: The Curiosity Gap Technique

**What You'll Learn:**
Create headlines and hooks that people can't ignore.

**Prompt Template:**
```
Create 5 headlines using the Curiosity Gap technique for [TOPIC].

Requirements:
- Tease a surprising insight without revealing it
- Make people want to click/read to learn more
- Include a specific number or benefit
- Keep under [CHARACTER LIMIT]

Examples of curiosity gap:
- "This one habit changed everything (and it's not what you think)"
- "I tried 100 productivity apps. Here's the one that actually works"
- "The $1 million mistake most entrepreneurs make"
```

**Real Example:**
```
Create 5 headlines using Curiosity Gap for AI prompts.

Requirements:
- Tease a surprising insight
- Make people want to learn more
- Include a specific number
- Keep under 60 characters

Examples:
- "This prompt hack saves 5 hours per week"
- "The AI prompt mistake costing you $10K/year"
```

**Expected Results:**
- 2-3x higher click rates
- More shares
- Better engagement

### Lesson 2.2: The Email Sequence Formula

**What You'll Learn:**
Create email sequences that convert cold prospects into customers.

**5-Email Launch Sequence:**

**Email 1 (Day 1): The Hook**
```
Subject: [CURIOSITY HOOK]

Body:
- Open with a surprising stat or question
- Introduce the problem
- Hint at the solution
- CTA: "Learn more" (no hard sell yet)
```

**Email 2 (Day 2): The Story**
```
Subject: [PERSONAL STORY]

Body:
- Tell a relatable story
- Show how others solved the problem
- Build credibility
- CTA: "See how it works"
```

**Email 3 (Day 3): The Proof**
```
Subject: [SOCIAL PROOF]

Body:
- Share testimonials
- Show results/numbers
- Address objections
- CTA: "Join them"
```

**Email 4 (Day 4): The Urgency**
```
Subject: [LIMITED TIME OFFER]

Body:
- Create scarcity (limited spots, time limit)
- Remind of benefits
- Address last objections
- CTA: "Get access now"
```

**Email 5 (Day 5): The Last Chance**
```
Subject: [FINAL CALL]

Body:
- This is the last email
- Offer expires soon
- Strong CTA
- Make it easy to buy
```

## MODULE 4: CONVERSION OPTIMIZATION

### Lesson 4.1: Landing Page Copy Framework

**What You'll Learn:**
Write landing page copy that converts 5-10% of visitors.

**The Framework:**
1. **Headline** - State the main benefit
2. **Subheadline** - Expand on the benefit
3. **Problem** - Identify the pain point
4. **Solution** - Present your offer
5. **Social Proof** - Show proof it works
6. **CTA** - Make it easy to buy

**Prompt Template:**
```
Write landing page copy for [PRODUCT].

Structure:
- Headline (under 10 words, clear benefit)
- Subheadline (expand on benefit)
- Problem statement (2-3 sentences)
- Solution (how your product solves it)
- 3 social proof points
- Strong CTA

Target audience: [AUDIENCE]
Price: [PRICE]
Main benefit: [BENEFIT]
Tone: [TONE]
```
